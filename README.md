User and Group Management API
Overview
Bu proje, kullanıcı ve grup yönetimi için modern bir .NET 9 Web API çözümüdür. Clean Architecture ve CQRS pattern'lerini kullanarak geliştirilmiştir.
🚀 Özellikler
👤 Kullanıcı Yönetimi
•	Kullanıcı CRUD işlemleri
•	Kullanıcı claim yönetimi
•	Kullanıcı grup yönetimi
•	Şifre sıfırlama ve unutulan şifre işlemleri
•	Profil fotoğrafı güncelleme
👥 Grup Yönetimi
•	Grup CRUD işlemleri
•	Grup claim yönetimi
🛠 Teknolojiler & Mimari
•	.NET 9
•	Clean Architecture
•	CQRS & MediatR
•	Entity Framework Core
•	AutoMapper
•	JWT Authentication
